# Screenshots

Drop your desktop + mobile screenshots of the iOS portfolio shell here, then
uncomment the image references in `../README.md`.

Suggested filenames:

- `desktop.png` — the iPad-framed home screen on a wide viewport
- `mobile.png` — the full-screen home screen on a phone viewport
- `widget.png` — the floating widget opened over the wallpaper
- `dock.png` — close-up of the glass dock with all three apps
